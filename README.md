# Educlaw 多学科 Agent

多Agent学习系统，支持英语、数学等学科。

## 功能

- 多学科支持（英语、数学）
- 智能路由：自动识别意图
- 上下文隔离：各科学记忆独立
- 积分激励系统

## 安装

```bash
cp -r . ~/.openclaw/agents/educlaw
openclaw gateway restart
```

## 使用

对话中说明想学什么学科：
- "学英语" → 英语Agent
- "学数学" → 数学Agent

## 学科配置

| 学科 | Agent | 状态 |
|------|-------|------|
| 英语 | 小英老师 | 可用 |
| 数学 | 小数老师 | 可用 |
| 语文 | 小语老师 | 即将推出 |

## 核心特点

### 拟人化教学
- 记住学生名字和兴趣
- 用学生喜欢的内容教学（游戏/足球/卡通）
- 情绪承接和鼓励

### 智能评分
从三个维度打分：
- 内容创意（0-10分）
- 词汇使用（0-10分）
- 语法正确（0-10分）

### 积分系统
- 完成练习获得积分
- 连续学习额外奖励
- 达到目标解锁奖励

## 配置

编辑 `router/config.yaml` 自定义路由规则。
